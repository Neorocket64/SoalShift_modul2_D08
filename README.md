<h3><a id="user-content-pembuatan" class="anchor" aria-hidden="true" href="#pembuatan"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Pembuatan</h3>
<p>Pertama, generate random string sampai 12 karakter menggunakan;</p>
<p><code>dd if=/dev/urandom|tr -dc A-Za-z0-9|head -c 12</code></p>
<p>yang akan disimpan melalui sebuah variabel.</p>
<p>Kemudian, supaya jenis karakter; <code>lowercase, uppercase, dan angka</code> ada dalam string tersebut,
membuat variabel masing-masing jenis karakter.</p>
<p>Diperiksa seluruh karakter dalam string, setiap ada salah satu jenis dari 3 tersebut, maka value variabelnya bertambah</p>
<p>Setelah itu, masuk bagian cek string pada file lain. Untuk ini, menambah variabel pembanding yang mengambil nilai file lain dalam loop.</p>
<p>Loop ini berisi salah satu variabel, yaitu <code>i</code>, untuk menyimpan angka yang ditujukan pada file lain <code>text"i".txt</code> -&gt; <code>text1.txt</code>. Jika tidak menemukan file dalam angka tersebut, maka akan dibuatkan file dengan angka tersebut.</p>

<span class="pl-en">chr2ascii</span>() {
	LC_CTYPE=C <span class="pl-c1">printf</span> <span class="pl-s"><span class="pl-pds">'</span>%d<span class="pl-pds">'</span></span> <span class="pl-s"><span class="pl-pds">"</span><span class="pl-smi">$1</span><span class="pl-pds">"</span></span>

<p>Ada 3 percabangan untuk selanjutnya,</p>
<ul>
<li>Jika alfabet lowercase</li>
<li>Jika alfabet Uppercase</li>
<li>Jika bukan alfabet</li>
</ul>
<p>Khusus untuk 2 pertama, menggunakan variabel yang berisi hasil konversi <code>chr2ascii</code>;</p>
<ol>
<li>Mengurangi value dengan (ascii alfabet pertama jenis tersebut - 1)</li>
<li>Value ditambah dengan jam saat logging</li>
<li>(sesuai syarat) Jika urutan alfabet sudah melebihi <code>z</code>, maka balik dari <code>a</code> (- 25)</li>
<li>Value ditambah lagi dengan (ascii alfabet pertama jenis tersebut - 1)</li>
<li>Merubah kembali menjadi karakter menggunakan <code>ascii2chr</code></li>
</ol>
<p>Karakter akan dimasukkan menuju sebuah variabel.</p>
<p>Proses ini berlangsung hingga akhir dari file tersebut.</p>
